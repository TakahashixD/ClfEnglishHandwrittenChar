# ClfEnglishHandwrittenChar
Discipline: 9800/01 - TOP. EM INTELIGENCIA COMPUTACIONAL

Link for the Images dataset: https://www.kaggle.com/datasets/dhruvildave/english-handwritten-characters-dataset
